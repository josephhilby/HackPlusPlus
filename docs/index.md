## About the Project

**Note:** This project is currently incomplete. It has been published in an attempt at learning in public. It 
represents my current understanding of the subject and not the finished state. I hope publishing this serves as a
benefit to others in their learning journey.

Hack++ is an extended computer platform that explores how to build a complete programmable machine from 
a single primitive digital logic gate (NAND). Beginning with the construction of basic logic gates, the project 
incrementally builds the memory system, CPU, and supporting software toolchain needed to execute programs. It serves as 
a systems-level demonstration of the relationship between logic, hardware architecture, and software execution.
